#  ProCredit Holding: Financial Performance Analytics

## Project Overview
This project presents a financial analysis of ProCredit Holding (2021 - Q3 2025). It tracks key performance indicators (KPIs) to evaluate profitability and operational efficiency.

## Key Results for North Macedonia (Sep-2025)
* **ROE (Return on Equity):** 10.2%
* **NIM (Net Interest Margin):** 2.7%
* **CIR (Cost-Income Ratio):** 59.2%
* **NPL Ratio:** 1.5% (Low risk)

## Tech Stack
* **Language:** Python (Pandas) for data processing
* **Visualization:** Power BI & Lovable
* **Data Source:** Quarterly Financial Reports
